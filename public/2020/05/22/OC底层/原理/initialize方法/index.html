objc_msgSend()

打开 runtime 源码 objc4-781，搜索 objc_msgSend()，会发现 objc_msgSend() 方法的源码是汇编语言。

objc_msgSend() - 类方法调用流程：
isa -> 类对象/元类对象，寻找方法，调用
superclass -> 类对象/元类对象，寻找方法，调用
superclass -> 类对象/元类对象，寻找方法，调用
