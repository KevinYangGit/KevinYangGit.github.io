Object 和 dynamic 的区别

Dart 中所有类型都是 Object 的子类，包括 Function 和 Null。

  • dynamic 与 Object 声明的变量都可以赋值任意对象，且后期可以改变赋值的类型。

  • Object 对象使用的属性和方法若不存在，编译时会报错。

  • dynamic 对象使用的属性和方法若不存在，编译时不会报错，运行时会报错。
