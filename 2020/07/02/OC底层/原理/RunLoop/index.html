什么是 RunLoop

  • 顾名思义
    运行循环，在程序运行过程中循环做一些事情。

  • 应用范畴

  1. 定时器（Timer）、PerformSelector
  2. GCD Async Main Queue
  3. 事件响应、手势识别、界面刷新
  4. 网络请求
  5. AutoreleasePool

RunLoop 的基本作用

  • 保持程序的持续运行
  • 处理App中的各种事件（比如触摸事件、定时器事件等）
  • 节省CPU资源，提高程序性能：该做事时做事，该休息时休息
